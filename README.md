  <h2>降水量の比較</h2>
  <p>京都府舞鶴市、大阪府茨木市、福井県福井市の三都市の2025年の月別の降水量をまとめる。
     気象庁のオープンデータから読み取った情報をグラフにし、下記に示す。</p>
  <p>  <img src="./kisyotyo_kosuiryo.png">  </p>

  <h2>考察</h2>
  <p>
  福井市は日本海側気候の影響により冬季降水量が多く、年間降水量も3都市中最大となった。一方、茨木市は瀬戸内海式気候の特徴から冬季降水量が少なく、梅雨や台風の時期に降水量が集中した。舞鶴市は両者の中間的な傾向を示した。地域ごとの降水量の違いは、地形や季節風の影響による気候特性の差が主な要因であると考えられる。
